== Description ==

My Online Shop Product is a clean, responsive and customizable WordPress theme, was designed to look awesome on any device.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and upload File or search for "My Online Shop Product"
3. Click install and activate to use your new theme right away.

== Frequently Asked Questions ==

= Is My Online Shop Product Free? =
Yes! My Online Shop Product is a free theme, and always will be a free theme.

= Does My Online Shop Product have a pro version? =
Currently it doesn't have any pro version. We're working on it.

= Where can I find the theme options? =
All of our options can be found in the Customizer in 'Appearance > Customize'.

= Does this theme have any widget areas? =
Yes. My Online Shop Product has 4 widget areas which you can add widgets to in Appearance > Widgets.

= How can I make my site look like your screenshot? =
If you want to replicate the screenshot you see on WordPress.org, follow the instructions available on [our official documentation](https://nbmblueeye.gitbook.io/my-online-shop-product/).

= Are there demo file available for My Online Shop Product? =
Yes, You can download demo file which are available on: https://github.com/nbmblueeye/my-online-shop-product-wordpress-theme-demo.git
